# Fuzzy Logic Controller

## Features

- **Fuzzy Logic Control:** Implements a fuzzy logic system to determine robot speed and steering based on distance and heading error.
- **Object Seeking:** Prioritizes navigation towards detected objects. (being fine tuned )
- **Path Navigation:** Follows planned paths while considering obstacles and exploration goals. (TBD) 
- **Spin Override:** Handles sharp turns and obstacle avoidance by temporarily reducing speed and focusing on rotation. (TBD)
